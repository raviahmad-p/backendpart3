# REST API Express PostgreSQL

## Deskripsi
Project ini adalah REST API sederhana menggunakan ExpressJS dan PostgreSQL untuk mengelola data user dengan fitur CRUD.

## Endpoint

- GET /api/users
- GET /api/users/:id
- POST /api/users
- PUT /api/users/:id
- DELETE /api/users/:id

## Cara Menjalankan

```bash
npm install
node index.js
